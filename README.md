# 👨‍💻 Edouard Lesieur - Portfolio

Welcome to the source code of my personal portfolio! This website serves as a showcase of my projects, skills, and experience in **Data Science, Artificial Intelligence, and MLOps**. 

Built with modern web technologies, this portfolio is fully responsive, bilingual (English/French), and features a clean, interactive UI.

## 🚀 Live Demo
🌍 **[https://edouardlesieur.com](https://edouardlesieur.com)**

## 🛠️ Tech Stack

This project is built with a focus on performance, maintainability, and user experience:

- **Framework**: [React 19](https://react.dev/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/) for robust and type-safe code.
- **Styling**: Modern CSS3 with custom variables, animations, and responsive design.
- **Internationalization (i18n)**: `react-i18next` for seamless switching between English and French.
- **Animations & UI**: `tsparticles-slim` for the interactive background and custom hooks for scroll reveal effects.
- **Icons**: `react-icons` (Feather Icons).

## 📂 Project Structure

- `public/` - Static assets, images, and `locales/` for i18n JSON translation files.
- `src/components/` - Reusable UI components (Buttons, Cards, Modals).
- `src/sections/` - Major page sections (Hero, About, Projects, Blog, Contact).
- `src/data/` - Content data like project lists and blog posts.
- `src/hooks/` - Custom React hooks (e.g., `useScrollReveal`).

## 💻 Running Locally

If you want to run this project on your local machine:

1. **Clone the repository**
   ```bash
   git clone https://github.com/Edouard21/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
